About Me

Hi! I'm Sebastián Martínez de la Rosa. I started programming when the pandemic started, and it was thanks to a class I had at university that I discovered this world, since then, all I want to do is keep learning and discovering new technologies.

I started with basic Python courses, and over time I began to learn about the different areas in which I could specialize, I chose web development.

My first framework was Flask, and with it I learned a bit of HTML and CSS. But I felt that my knowledge was very limited, so I continued studying until I found the different courses that Harvard offers, the first one I took was to learn the fundamentals of programming, and the second was focused on web development with Django, SQL, Javascript, HTML and CSS.

I have done several projects, many related to the courses, but others are real projects, the most recent is a promotion page for a school. Now I'm working on something bigger, an "encyclopedia for programmers".

Here are a few technologies I’ve been working with recently:

  • Python (Django, Flask)
  • JavaScript (React)
  • SQL (MySQL, SQLite)
  • HTML5
  • CSS3 (Bootstrap, Tailwind)
  • Git (Github)
  • Linux (Ubuntu)
  • Nginx
  • Gunicorn
