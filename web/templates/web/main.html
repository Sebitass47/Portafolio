Acerca de mi

Hola!, Soy Sebastián Martínez de la Rosa. Empecé a programar cuando la pandemia empezó, y fue gracias a una clase que tuve en la universidad que descubrí este mundo, desde entonces, lo único que quiero hacer es seguir aprendiendo y descubriendo nuevas tecnologías.

Empecé con cursos básicos de Python, y con el tiempo empecé a conocer las diferentes áreas en las que me podía especializar, escogí desarrollo web.

Mi primer framework fue Flask, y con el aprendí un poco de HTML y CSS. Pero sentía que mis conocimientos era muy limitados, así que seguí estudiando hasta que encontré los diferentes cursos que ofrece Harvard, el primero que curse fue para aprender los fundamentos de la programación, y el segundo fue enfocado al desarrollo web con Django, SQL, Javascript, HTML and CSS.

He realizado varios projectos, muchos relacionados con los cursos, pero otros son proyectos reales, el más reciente es una página de promoción para una escuela. Ahora estoy trabajando en algo más grande, una "enciclopedia para programadores".

Aquí hay algunas tecnologías con las que he estado trabajando recientemente:

  • Python (Django, Flask)
  • JavaScript (React)
  • SQL (MySQL, SQLite)
  • HTML5
  • CSS3 (Bootstrap, Tailwind)
  • Git (Github)
  • Linux (Ubuntu)
  • Nginx
  • Gunicorn
